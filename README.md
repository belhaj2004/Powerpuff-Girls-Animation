# Powerpuff Girls Animation

A creative animation inspired by the Powerpuff Girls, built using the Processing framework with Java.

## About
- Character-based 2D animation
- Built entirely with Processing (Java)
- Demonstrates object-oriented design for visual elements

## How to Run
1. Download [Processing IDE](https://processing.org/download)
2. Open `p2b_modeling.pde`
3. Click the ▶ Run button
